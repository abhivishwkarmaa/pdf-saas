# Roadmap: PDF SaaS

**Milestone:** v1.0 — Full free catalog
**Phases:** 8

## Phase 1: Foundation & catalog shell

**Goal:** Runnable Next.js app, infra, tool registry, upload API, marketing UI

## Phase 2: Light PDF operations (sync)

**Goal:** Merge, split, pages, rotate, PDF↔images

## Phase 3: Async job pipeline

**Goal:** BullMQ, worker, job UI, TTL cleanup

## Phase 4: Optimize PDF

**Goal:** Compress, PDF/A, repair

## Phase 5: Office conversions

**Goal:** PDF↔Office, HTML→PDF, image→Word

## Phase 6: Edit & security

**Goal:** Watermark, page numbers, crop, protect/unlock, redact, sign

## Phase 7: Intelligence

**Goal:** OCR, compare PDF

## Phase 8: Launch polish

**Goal:** SEO, health checks, deploy docs, enable all tools
